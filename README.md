**Shazwan Coffee Shop**

A modern coffee shop website built using HTML, CSS, and JavaScript. This project was created as a frontend development practice project featuring a responsive and user-friendly design.


**About the Project**

Shazwan Coffee Shop is a website that showcases information about a coffee shop, including its menu, products, location, and contact details.

This project was developed to improve web development skills using fundamental frontend technologies.


**Technologies Used**

* HTML5
* CSS3
* JavaScript
* Google Fonts
* Font Awesome


**Features**

* Responsive Design
* Navigation Bar
* Hero Section
* About Section
* Menu Section
* Product Section
* Contact Section
* Social Media Links
* Smooth Scrolling
